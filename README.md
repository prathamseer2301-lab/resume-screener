# AI Resume Screener

AI-powered resume screener that scores and ranks CVs against a job description using Google Gemini AI.

## Features
- Upload multiple CVs in PDF format
- Paste any job description
- AI scores each CV from 0-100
- Shows strengths, weaknesses and verdict for each candidate
- Ranks candidates automatically

## Tech Stack
- Python 3.12
- Google Gemini AI (gemini-2.5-flash)
- Streamlit
- pdfplumber

## How to run
streamlit run app.py
